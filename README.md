# United Contractors Organisation (UCO) — Website

A single-page, responsive site built from the UCO brochure (navy/gold institutional design, hero, About, Vision/Mission/Values, What UCO Offers, Membership & Global Network, and Contact footer).

## Files
- `index.html` — the entire site (HTML/CSS/JS, no build step, no dependencies except Google Fonts)
- `assets/uco-logo.jpg` — the UCO crest logo used in the nav, hero, and footer

## Before you launch
- Swap the 4 gradient "photo" placeholders in the "What UCO Offers" section for real site photography
- Update social links in the footer (LinkedIn/Twitter/Facebook/YouTube currently point to `#`)
- Replace the QR placeholder icon with a real QR code image pointing to www.ucoindia.org if desired

See the bottom of this repo's chat thread for full GitHub Pages deployment steps.
